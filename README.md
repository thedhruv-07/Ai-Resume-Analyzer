# AI Resume Analyzer 🚀

AI-powered platform that analyzes resumes against job descriptions and gives ATS score, keyword gaps, and improvement suggestions.

## 🔥 Features
- ATS Score Generation
- Keyword Gap Analysis
- AI Suggestions for Resume Improvement
- PDF Resume Upload & Parsing

## 🧠 Tech Stack
React.js, Node.js, Express.js, MongoDB  
OpenAI API / Groq API  

## ⚡ How it works
1. Upload resume (PDF)
2. Add job description
3. AI analyzes and returns insights
